Разные платформы

PWA:

  • Это сайт
  • Устанавливается
  • Выглядит как нативное
  • Работает оффлайн
  • Присылает Push-уведомлениям
  • И другие Web API

Для бизнеса:

  • Кроссплатформенность
  • Индексируется
  • Маленький вес (~2-5 МБ)
  • Установку предлагает браузер
  • Установка моментальная

Веб манифеста

JSON-файл, который влияет на то, как выглядит, воспринимается и ведёт себя приложение.

Пример веб манифеста

    {
      "name": "Прогрессивные Веб Приложения",
        "short_name": "PWA",
      "start_url": "/",
        "display": "fullscreen",
      "icons": [...]
    }

Для Safari есть набор метатегов

    <meta name="apple-mobile-web-app-capable"
      content="yes">
    <meta name="apple-mobile-web-app-title"
      content="PWA">
    <meta name="apple-mobile-web-app-status-bar-style"
      content="black">
